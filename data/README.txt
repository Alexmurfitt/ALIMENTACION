# NutriApp 🍏

Sistema de seguimiento de alimentos, macronutrientes y micronutrientes.  
Hecho en Python + Flask + MySQL.

## Funcionalidades:
- Registro de alimentos
- Cálculo de nutrientes por gramos
- Visualización de resumen nutricional
